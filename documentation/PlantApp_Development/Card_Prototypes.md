# Plant Card Prototypes — v0.1

> 5 карточек-прототипов для отработки структуры.
> Цель: понять, какие секции нужны каждому типу, зафиксировать формат.
> Это НЕ база данных — это макет контента.

---

## Общая структура карточки

```
┌─────────────────────────────────┐
│  🖼 Фото (треть экрана)         │
├─────────────────────────────────┤
│  Название + семейство           │
│  [Badges row — tappable]        │
├─────────────────────────────────┤
│  СЕКЦИИ LEVEL 1                 │
│  (зависят от типа растения)     │
├─────────────────────────────────┤
│  ▼ Advanced (Level 2)           │
└─────────────────────────────────┘
```

### Badges (верхняя полоса, tappable → scroll к секции)
- **Difficulty**: Easy / Medium / Advanced + цвет (зелёный/жёлтый/красный)
- **Toxicity**: "Pet Safe" (зелёный) / "Toxic: sap, leaves" (красный с уточнением)
- **Edible**: для съедобных — "Edible: leaves" / "Edible: fruit" (зелёный)
- **Water**: Low / Medium / High + warning icon если overwater-sensitive
- **Light**: Low / Bright / Full Sun
- **Lifecycle**: "Lives 2+ years" / "Annual — regrow from seed"

# 🟢 1. Crassula ovata — Jade Plant (Денежное дерево)
**Type: DECORATIVE**

### Badges
`Easy` `Toxic: pets` `Water: Low ⚠️ overwater kills` `Bright light` `Lives 20+ years`

### About
Неубиваемый суккулент, растёт десятилетиями. В хороших условиях превращается в мини-дерево с толстым стволом. Идеален для начинающих — прощает забытый полив, но не переживёт перелив. Символ удачи и денег.

**Difficulty: Easy** — забудь полить на 2 недели, ему норм. Залей — сгниёт.

### Care

**Вода**
- Лето: раз в 2-3 недели, когда земля полностью сухая
- Зима: раз в месяц
- ⚠️ WARNING: Перелив — убийца #1. Листья стали мягкими и прозрачными = гниль корней. Лучше забыть полить, чем перелить.
- Polivalka: автоматически по датчику, start 15% / stop 35%

**Свет**
- Яркий рассеянный или прямое солнце
- Терпит полутень, но растёт медленнее и вытягивается
- ⚠️ В Дрездене зимой: досветка желательна, но выживет и без неё

**Почва и горшок**
- Суккулентный грунт с песком, хороший дренаж
- Любит тесный горшок — не спеши пересаживать
- Пересадка раз в 2-3 года
- Горшок 15-20 см → ожидай высоту до 60-90 см

**Удобрение**
- 2-3 раза за сезон, разбавленное
- Только весна-лето

### Toxicity
- 🐱🐶 Токсично для кошек и собак при поедании
- 👤 Для людей безопасно
- Что делать: держать повыше если есть питомцы

### Advanced (Level 2)
- PPFD: 200-600 µmol/m²/s | DLI: 8-20 mol/m²/day
- Температура: 15-24°C, переживёт кратковременно до 5°C
- Air Humidity: 30-50% (сухой воздух — норм)
- pH почвы: 6.0-7.0
- Sensor Settings: start 15% / stop 35% (Succulents preset)
- Tips: можно формировать обрезкой как бонсай. Опавшие листья укореняются сами.

# 🟢 2. Phalaenopsis amabilis — Moth Orchid (Орхидея)
**Type: DECORATIVE**

### Badges
`Medium` `Pet Safe` `Water: Medium` `Bright indirect` `Lives 10+ years`

### About
Самая неприхотливая из орхидей. Цветёт 2-3 месяца за раз, при правильном уходе — повторно. Не сажать в землю! Живёт в коре. Прозрачный горшок — не для красоты, корням нужен свет.

**Difficulty: Medium** — не сложно, но есть нюансы: полив в кору, нельзя в обычный грунт, нужен перепад температур для цветения.

### Care

**Вода**
- Лето: раз в 7-10 дней, замочить кору на 15 мин, дать стечь
- Зима: раз в 10-14 дней
- ⚠️ WARNING: Вода в розетке листьев = гниль. Поливать только корни/кору!
- ⚠️ WARNING: Корни зелёные = влажные, серебристые = пора поливать
- Polivalka: start 25% / stop 50% (калибровать под кору!)

**Свет**
- Яркий рассеянный, БЕЗ прямого солнца
- Восточное окно — идеал
- ⚠️ Прямое солнце = ожоги на листьях (жёлтые/белые пятна)
- ⚠️ В Дрездене зимой: досветка поможет с повторным цветением

**Почва и горшок**
- НЕ ЗЕМЛЯ! Кора, сфагнум, перлит — orchid mix
- Прозрачный горшок — корни фотосинтезируют
- Пересадка раз в 2-3 года, когда кора разложилась
- Горшок 12-15 см достаточно

**Удобрение**
- Специальное орхидейное, раз в 2 недели
- Только весна-осень
- Правило: "weakly, weekly" — слабая концентрация, но часто

### Toxicity
- ✅ Безопасна для людей и животных
- Никаких мер предосторожности

### Advanced (Level 2)
- PPFD: 100-300 µmol/m²/s | DLI: 4-12 mol/m²/day
- Температура: 18-27°C днём, для цветения нужен ночной перепад на 5-7°C
- Air Humidity: 50-70%
- Sensor Settings: start 25% / stop 50% (Tropical preset, калибровать для коры)
- Tips: после опадания цветов — обрезать стебель над узлом для повторного цветения. Воздушные корни вне горшка — это нормально, не запихивать обратно.

# 🟡 3. Ocimum basilicum — Basil (Базилик)
**Type: GREENS**

### Badges
`Easy` `Pet Safe` `Edible: leaves` `Water: High ⚠️ не мочить листья` `Full Sun` `Annual`

### About
Быстрорастущая пряная трава для кухни. Идеален для пасты, салатов, песто. Чем больше срываешь — тем пышнее растёт. Однолетник: живёт сезон, потом сеешь заново. Аромат усиливается на солнце.

**Difficulty: Easy** — растёт быстро, прощает ошибки. Главное: солнце и вовремя щипать цветы.

### Care

**Вода**
- Лето: каждые 3-5 дней, почва влажная но не мокрая
- Зима: каждые 5-7 дней
- ⚠️ WARNING: Не мочить листья! Мокрые листья → чёрные пятна (грибок)
- ⚠️ WARNING: Пересушил → завял → быстро полей, обычно оживает
- Polivalka: start 40% / stop 65% (Herbs preset)

**Свет**
- Полное солнце 6+ часов (южное окно)
- ⚠️ WARNING: Без достаточного солнца вытягивается, теряет вкус и аромат
- ⚠️ В Дрездене зимой: БЕЗ досветки не вырастет нормально. Нужна фитолампа или сей весной.

**Почва и горшок**
- Питательный грунт с компостом, хороший дренаж
- Горшок от 15 см, можно несколько кустов в один
- Однолетник — пересадка не нужна, сей новый

**Удобрение**
- Азотное жидкое, раз в 2-3 недели
- Весь сезон роста
- ⚠️ Для съедобных: только органические удобрения!

### Toxicity
- ✅ Полностью безопасен — это еда
- Безопасен для животных

### 🌿 Harvest (секция ТОЛЬКО для greens/fruiting)

**Что собирать:** листья и верхушки побегов

**Когда начинать:** когда куст 15+ см и имеет 3+ пары листьев

**Как правильно:**
- Срывать/срезать верхушки побегов (верхние 2 листа + точку роста)
- Это заставляет ветвиться → куст становится пышнее
- НЕ обрывать нижние листья — они кормят растение
- Собирать утром — максимум аромата и эфирных масел

**Как часто:** каждые 5-7 дней, не больше 1/3 куста за раз

**Для чего:**
- Свежие листья: салаты, капрезе, брускетта
- Песто: снять много листьев разом, пробить с чесноком/кедровыми/пармезаном
- Заморозка: сорвал → помыл → в пакет в морозилку

**⚠️ Цветение = конец вкуса**
Как только появились бутоны — НЕМЕДЛЕННО прищипни! После цветения листья становятся горькими. Если пропустил — собери семена для следующего сезона.

### Advanced (Level 2)
- PPFD: 300-600 µmol/m²/s | DLI: 12-22 mol/m²/day
- Температура: 18-27°C (не переносит холод ниже 10°C)
- Air Humidity: 40-60% (не капризен)
- Спектр досветки: синий + белый (вегетативный рост, листовая масса)
- Sensor Settings: start 40% / stop 65% (Herbs preset)
- Tips: эфирные масла усиливаются на солнце. Можно выращивать рядом с томатами — взаимовыгодное соседство (companion planting).

# 🟡 4. Salvia rosmarinus — Rosemary (Розмарин)
**Type: GREENS**

### Badges
`Easy` `Pet Safe` `Edible: leaves, stems` `Water: Low ⚠️ overwater kills` `Full Sun` `Perennial`

### About
Средиземноморский вечнозелёный кустарник. Пахнет невероятно, на кухне незаменим. Живёт годами, со временем одревесневает. Любит бедную почву и сухость — чем хуже условия (в меру), тем ароматнее. Полная противоположность базилика по поливу.

**Difficulty: Easy** — в целом неубиваемый, но зимой в квартире может капризничать из-за сухого воздуха и нехватки света.

### Care

**Вода**
- Лето: раз в 7-10 дней, когда верхний слой просох
- Зима: раз в 10-14 дней, чуть суше
- ⚠️ WARNING: Перелив = корневая гниль = смерть. Розмарин — средиземноморец, привык к засухе.
- ⚠️ WARNING: Глиняный горшок (терракота) лучше пластика — дышит, сохнет быстрее
- Polivalka: start 20% / stop 40% (Herbs preset, сдвинуть к сухому)

**Свет**
- Полное солнце 6+ часов
- ⚠️ WARNING: Без солнца зимой → вытягивается, теряет аромат, листья сохнут
- ⚠️ В Дрездене зимой: досветка НУЖНА для нормального роста. Или поставить на самый светлый подоконник и смириться с замедлением.

**Почва и горшок**
- Бедная, песчаная, хорошо дренированная. Слегка щелочная.
- Терракотовый горшок — идеал
- Пересадка раз в 1-2 года
- В горшке 20-25 см вырастет до 40-60 см

**Удобрение**
- Лёгкая подкормка раз в весну — и хватит
- Не перекармливать! Бедная почва = больше аромата
- ⚠️ Для съедобных: только органические

### Toxicity
- ✅ Полностью съедобен — это специя
- Безопасен для животных
- Бонус: отпугивает некоторых вредителей (companion plant)

### 🌿 Harvest

**Что собирать:** веточки с листьями (иголками)

**Когда начинать:** когда куст 15+ см и хорошо разветвился

**Как правильно:**
- Срезать веточки 5-10 см от верхушек
- Не обрезать больше 1/3 куста за раз
- Срезать НАД узлом — от него пойдут новые побеги
- Одревесневшие стебли тоже можно использовать (бросить в бульон целиком)

**Как часто:** по мере надобности, минимум раз в 2-3 недели для формирования

**Для чего:**
- Мясо (баранина, курица, свинина) — положить веточку при жарке/запекании
- Картофель — мелко нарезать иголки, смешать с маслом и солью
- Хлеб фокачча — воткнуть веточки в тесто перед выпечкой
- Чай/настой — заварить свежую веточку кипятком

**⚠️ Цветение = ОК!**
В отличие от базилика, цветение розмарина НЕ портит вкус. Цветы тоже съедобны и красиво смотрятся в салатах.

### Advanced (Level 2)
- PPFD: 300-600 µmol/m²/s | DLI: 12-22 mol/m²/day
- Температура: 10-24°C (переносит кратковременно до -5°C!)
- Air Humidity: 30-50% (предпочитает сухость)
- pH почвы: 6.0-7.5 (слегка щелочная)
- Sensor Settings: start 20% / stop 40% (Herbs preset, dry side)
- Tips: обрезка после цветения → пышный куст. Можно размножать черенками в воде.

# 🔴 5. Solanum lycopersicum — Tomato (Томат)
**Type: FRUITING**

### Badges
`Advanced` `Toxic: leaves, stems ⚠️` `Edible: fruit` `Water: High` `Full Sun` `Annual`

### About
Король домашнего огорода. Плоды — еда, листья и стебли — ядовиты (соланин). Требует много солнца, места, подкормки и внимания. Комнатный томат реалистичен только на балконе/подоконнике с досветкой и мелкоплодными сортами (черри). Полноразмерные сорта в квартире — утопия.

**Difficulty: Advanced** — требует: опыление вручную (нет пчёл дома), подвязку, пасынкование, много света, регулярную подкормку. Не для первого растения.

### Care

**Вода**
- Активный рост/плодоношение: каждые 2-3 дня, обильно
- Рассада: каждые 3-5 дней, умеренно
- ⚠️ WARNING: Поливать ТОЛЬКО под корень, не на листья (фитофтора!)
- ⚠️ WARNING: Неравномерный полив → трещины на плодах
- ⚠️ WARNING: Перелив → гниль корней. Недолив → вершинная гниль (чёрное дно плода)
- Polivalka: start 45% / stop 70% (Custom, зависит от фазы)

**Свет**
- Полное солнце 8+ часов — это минимум!
- ⚠️ WARNING: Мало света = мало плодов или вообще ничего. Без 8 часов даже не начинай.
- ⚠️ В Дрездене: реально только на южном балконе летом ИЛИ с мощной фитолампой (PPFD 400+)
- Для плодоношения нужен красный спектр!

**Почва и горшок**
- Питательный грунт с компостом и перлитом
- Горшок МИНИМУМ 20-30 см для черри, 40+ см для крупных сортов
- Глубина важна — корни идут вниз
- Можно заглублять стебель при посадке — даст дополнительные корни

**Удобрение**
- До цветения: азотное (рост зелёной массы)
- Цветение и плодоношение: калийно-фосфорное (плоды!)
- Раз в неделю в период активного роста
- ⚠️ Для съедобных: только органические! Кальций отдельно (против вершинной гнили)

### Toxicity / Safety
- 🍅 Плоды: СЪЕДОБНЫ (ради этого и выращиваем)
- ⚠️ Листья, стебли, незрелые зелёные плоды: ТОКСИЧНЫ (соланин)
- 🐱🐶 Токсично для кошек и собак (всё кроме спелых плодов)
- Что делать: мыть руки после работы с кустом, не давать питомцам жевать листья

### 🌿 Harvest

**Что собирать:** спелые плоды

**Как определить спелость:**
- Равномерный цвет (красный/жёлтый/оранжевый — зависит от сорта)
- Слегка мягкий при нажатии
- Легко отделяется от кисти
- ⚠️ Зелёные плоды = незрелые = содержат соланин. Можно дозреть на подоконнике, но вкус хуже.

**Как часто:** черри — каждые 2-3 дня в сезон, обычные — по мере созревания

**Для чего:**
- Черри: салаты, закуски, прямо с куста
- Обычные: соусы, салаты, запекание
- Избыток: вялить, заморозить, сделать соус впрок

### 📈 Growth Phases (секция ТОЛЬКО для fruiting)

| Фаза | Срок | Что происходит | Уход |
|------|------|----------------|------|
| **Рассада** | 0-4 нед | Из семян, первые листья | Тепло 22-25°C, свет 16ч/день, умеренный полив |
| **Вегетация** | 4-8 нед | Рост стебля и листьев | Азотные удобрения, подвязка, пасынкование |
| **Цветение** | 8-10 нед | Жёлтые цветки | Калий+фосфор, опыление (потрясти куст!), меньше азота |
| **Плодоношение** | 10-16 нед | Завязь → спелый плод | Обильный полив, кальций, собирать по мере спелости |
| **Конец сезона** | 16+ нед | Замедление, увядание | Собрать зелёные на дозревание, убрать куст, собрать семена |

**⚠️ Пасынкование:** боковые побеги из пазух листьев НАДО удалять (для крупноплодных). Черри — можно оставить, они кустовые.

**⚠️ Опыление:** дома нет ветра и пчёл. Потряси цветущую кисть или проведи кисточкой по цветкам. Без опыления = без плодов.

### Advanced (Level 2)
- PPFD: 400-800 µmol/m²/s | DLI: 20-35 mol/m²/day (очень много!)
- Температура: 18-27°C днём, 15-18°C ночью (перепад важен для завязи)
- Air Humidity: 50-70%
- Спектр: рассада = синий, вегетация = белый/синий, цветение/плоды = красный!
- pH почвы: 6.0-6.8
- Sensor Settings: start 45% / stop 70% (Custom, корректировать по фазе)
- Companion planting: базилик рядом отпугивает вредителей и (говорят) улучшает вкус

# Выводы из прототипов

## Что видно по 5 карточкам:

### Общие секции (все типы):
1. **Badges** (tappable)
2. **About** (actionable description + difficulty с пояснением)
3. **Care** (вода + свет + почва/горшок + удобрение, каждый с warnings)
4. **Toxicity / Safety** (что токсично, что безопасно — рядом)
5. **Advanced** (PPFD/DLI, температура, humidity, sensor settings, tips)

### Только Greens + Fruiting:
6. **Harvest** (что, когда, как, для чего)

### Только Fruiting:
7. **Growth Phases** (таблица фаз с уходом по каждой)

### Ключевые паттерны:
- **Warnings встроены в Care**, а не в отдельную секцию
- **Difficulty = badge + пояснение** ("Advanced because...")
- **Toxic + Edible рядом** (авокадо-паттерн: "плод съедобен, листья ядовиты")
- **Горшок ↔ размер** всегда вместе
- **Региональные предупреждения** (Дрезден зимой → досветка)
- **Удобрения для съедобных** — отдельное предупреждение про органику
- **Цветение** — разное значение: базилик (плохо!) vs розмарин (ок!) vs томат (нужно опылять!)
