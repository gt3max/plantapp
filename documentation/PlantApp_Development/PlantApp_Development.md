## 1. Анализ конкурентов

### 1.1 Planta (getplanta.com)

**Общее:** Шведское приложение, 7-10M пользователей, ~$4-8M/год, bootstrapped.
**Стек:** Native iOS (Swift), Android (Kotlin). Размер ~189 MB.
**Рейтинг:** iOS 4.8/5 (107K отзывов), Android 4.2/5.

**Монетизация:**
- Free: базовые напоминания, 5 растений, ограниченная ID
- Premium Monthly: $7.99-$9.99/мес
- Premium Annual: $35.99-$47.99/год
- Lifetime: нет
- Без рекламы. 7-дневный trial.

**Что за paywall:** AI-идентификация, диагностика болезней (Dr. Planta), светомер, кастомные расписания, журнал, shared care, безлимит растений.

**Жалобы пользователей (топ):**
1. Негибкие расписания полива — нельзя настроить частоту
2. Переполив суккулентов и орхидей (67% пользователей)
3. Агрессивный paywall — бесплатная версия бесполезна
4. Plant ID плохо работает на редких видах (~68% vs ~92% для обычных)
5. Путаница с биллингом (годовой платёж сразу, не помесячно)
6. Нет vacation mode
7. Нет выбора единиц (только Fahrenheit/inches)

**Сильные стороны:**
- Огромная база (25,000+ видов)
- Красивый UI/UX
- Plant ID по фото
- 30+ параметров для расписания полива (погода, сезон, свет)
- Community features

**Слабости = наши возможности:**
- 0 интеграций с железом/датчиками
- Угадывает полив алгоритмом, не измеряет
- Напоминает, но не поливает
- Нет оффлайна

### 1.2 Тестирование Planta — личный опыт (2026-03-14)

**Скриншоты:** `documentation/PlantApp_Development/Planta/IMG_4880-4916.PNG`

#### Onboarding и Paywall

**Первое впечатление:** невозможно использовать приложение без подписки. Каждая полезная функция (Identify, Dr Planta, Weather, Light Meter) ведёт к paywall-экрану.

**Paywall-стратегия Planta:**
- 4 разных paywall-экрана (carousel upsell) — каждый продаёт отдельную фичу
- Годовая подписка: "$3/mo" крупно, "$35.99 billed at once" мелко — пользователи жалуются на обман
- Месячная: $9.99 — заведомо невыгодная, чтобы подтолкнуть к годовой
- "YOUR ONE TIME OFFER" — 30% скидка ($24.99/год) — FOMO-давление
- Trial 7 дней — нигде внутри приложения не виден, настройки подписки спрятаны
- Кнопка "Manage Subscription" отсутствует в Settings

**Вывод:** Planta = энциклопедия за paywall. Бесплатная версия бесполезна.

#### Структура профиля растения (Monstera deliciosa)

**Header:**
- Фото-карусель (swipe)
- Название: Monstera + Monstera deliciosa (латынь)
- Fun fact (зелёный блок)
- Quick tags: `Easy` `Part sun` `Medium` `Toxic`

**Tab-навигация (10 табов, горизонтальный скролл):**

| Tab | Данные | Полезность |
|-----|--------|-----------|
| Water | Частота (every 15 days in March — сезонная!), demand level | Высокая |
| Light | Preferred + Also suitable | Высокая |
| Humidity | Need level, частота опрыскивания | Средняя |
| Fertilizing | Тип, частота, рекомендованные удобрения (фото) | Средняя |
| Soil & Repotting | Тип почвы, частота пересадки, чистка листьев | Средняя |
| Toxicity | Для кого токсично (люди/животные), предупреждение | Высокая |
| Characteristics | Lifecycle, Dimensions, Goals, Leaf type | Низкая (энциклопедия) |
| Temperatures | Ideal range, Indoor/Outdoor months по геолокации | Средняя |
| Problems & Pests | 8 проблем + 5 вредителей, Sensitive to draft | Высокая |
| Identity | Scientific name, Common names, Report error | Низкая (энциклопедия) |

**Оценка:** Сделано подробно, но избыточно. 10 табов — information overload. Большинство данных (Lifecycle: Perennial, Dimensions: 12-450cm, Botanical nomenclature) — для ботаников, не для обычного пользователя.

#### Хорошие идеи (взять)

- Tab-навигация по категориям — удобнее чем один длинный скролл
- Сезонная адаптация: "every 15 days **in March**" — частота привязана к месяцу
- Геолокация для температуры: "outdoor months in Дрезден" — персонализация
- Quick tags в header (Easy / Part sun / Toxic) — моментальное понимание
- Problems & Pests как отдельный раздел
- Placement: Indoor (6 типов) + Outdoor (7 типов) + Custom — влияет на расписание
- Light settings: 4 уровня (Full sun / Part sun / Shade / Dark) + Light meter

#### Избыточное (не брать)

- 10 табов — слишком много, 90% пользователей не листают дальше 3-го
- Botanical nomenclature, Lifecycle, Dimensions — не помогает ухаживать
- "Learn about fertilizing", "Lifecycle of plants" — ссылки на статьи, не контент
- Fun facts — мило, но не функционально
- Growing goals ("Decorative", "Air purifier") — очевидно для владельца

#### Отдельные разделы

- **Notifications:** Push on/off, время напоминания (10:00), Weather alerts = Premium
- **Inbox:** пустой, "No messages yet" — непонятная фича
- **Explore:** Browse issues (Pests/Leaves/Stems), каталог болезней с фото — полезно
- **Light Meter:** Premium, но ручной выбор бесплатно (4 уровня)
- **Dr Planta:** AI-чат для диагностики — полностью за paywall

#### Отзывы пользователей (исследование App Store, Reddit, Apple Community)

**Топ жалобы:**

| Проблема | Частота | Серьёзность |
|----------|---------|------------|
| Всё за paywall, бесплатная версия бесполезна | Очень высокая | Средняя |
| Кнопка отмены подписки не работает | Средняя | Высокая |
| "$3/mo" а списывают $36 сразу | Средняя | Высокая |
| Trial списывает после отмены | Средняя | Высокая |
| Нет одноразовой покупки | Средняя | Низкая |
| Premium не стоит денег | Средняя | Средняя |
| Identification неточная (3 из 5 ошибок) | Низкая | Средняя |

**Цитаты:**
- *"There's nothing in premium I can't find somewhere else for free"* — Google Lens делает то же
- *"Misled to think my subscription would be $3/month. Instead, $35.99 was debited immediately"*
- *"Cancel subscription button flashed but went nowhere"* (Apple Community)
- *"Misidentified 3 of 5 plants"*

**Спрятанные настройки аккаунта (dark pattern):**
- На главном экране НЕТ иконки настроек (стандартная шестерёнка)
- Путь: Колокольчик → Inbox → Шестерёнка (внутри Inbox!) → Newsfeed settings
- 3 уровня вложенности — и это настройки УВЕДОМЛЕНИЙ, не аккаунта
- Управление подпиской, статус trial, email, logout — отсутствуют полностью
- "Offers" включены по умолчанию (маркетинговые push)
- Weather alerts и Dr Planta tasks помечены "Premium" — upsell даже в настройках

**Юридический риск:** FTC (2024) выпустила исследование о dark patterns в subscription apps. Сломанная кнопка отмены + спрятанные настройки Planta — потенциальное нарушение

## 2. Стратегия дифференциации

### Наше ключевое преимущество

| Planta (software-only) | PlantApp + Polivalka (hardware + software) |
|------------------------|---------------------------------------------|
| Угадывает когда поливать | Знает (реальный датчик) |
| Напоминает — человек поливает | Поливает автоматически |
| $36-48/год подписка | Одноразовая покупка устройства |
| Переполив суккулентов | Невозможно — датчик останавливает |
| Нет оффлайна | Полный оффлайн (AP mode) |
| 0 железа | Мы и есть железо |

### Позиционирование

**Рабочий слоган:** "Why guess when you can measure?"

### Принципы PlantApp (из анализа Planta)

#### Принцип 1: "Информация = Действие"
Любой совет по растению сводится к конкретному действию или включению оборудования.
Не показывать информацию ради информации (Botanical nomenclature, Lifecycle type).

| Информация | Действие | Оборудование |
|---|---|---|
| "Полив каждые 15 дней" | Включить насос | Polivalka |
| "Part sun, part shade" | Совет куда поставить / вкл-выкл лампу | Фитолампа |
| "18-25°C" | Предупреждение если холодно | Термостат / push |
| "High humidity" | Опрыскать / включить увлажнитель | Humidifier / push |
| "Удобрение раз в месяц" | Push напоминание | — |
| "Toxic" | Предупреждение при добавлении | Push (если есть pets) |

#### Принцип 2: "Два уровня" (как Sensor Mode + Advanced)
- **Level 1 (основной экран):** Water, Light, Toxicity, Top-3 problems — для 90% пользователей
- **Level 2 (Advanced / Full care guide):** Fertilizing, Soil, Temperature zones, Pests, Humidity — для продвинутых
- НЕ показывать 10 табов сразу. Один экран + drill-down

#### Принцип 3: "Честная монетизация"
- Free tier с реальной функциональностью (не пустышка)
- Identify бесплатно (3-5 раз/день)
- Базовая диагностика бесплатно
- Погода бесплатно (OpenWeatherMap free tier)
- Manage Subscription прямо в Settings (не прятать!)
- Чёткий trial: "7 days free, NO charge until day 8"
- Честная цена: "$30/year" без мелкого шрифта "$2.50/mo"
- Рассмотреть lifetime plan ($49.99?)
- Подписка = AI-фичи (живой контент), статика = бесплатно

#### Принцип 4: "Группировка растений" (killer feature)
Planta: каждое растение живёт отдельно. Реальность: растения стоят группами.

**Site = группа с общими условиями:**
- Общий свет, температура, влажность (условия места)
- Индивидуальный полив (каждая Polivalka на своём горшке)
- Совместимость — предупреждение если растения конфликтуют
- Пример: "Basil needs more sun than Monstera — consider moving"

**Use case: плодовые растения (помидоры, перцы, клубника)**
- Advanced режим: фазы вегетации → разные режимы полива/удобрений
- Рассада → Цветение → Плодоношение = разные потребности
- Влияет на вкус и урожай — здесь подробные настройки оправданы

#### Принцип 5: "Try before you buy"
Planta блокирует всё до оплаты → огромный churn на этапе активации.
PlantApp: дай попробовать → покажи ценность → потом предложи Premium.
Один ненавязчивый upsell когда лимит исчерпан (не 4 paywall-экрана подряд)
